💰 Importar Custódia RF

Importe o arquivo Custódia.xlsx para analisar a exposição de Renda Fixa dos seus clientes.

Formato: YYYY-MM (exemplo: 2024-10)

Aceita apenas .xlsx (arquivo Excel da Custódia RF).

💡 Como usar:

  • Baixe o arquivo Custódia.xlsx da XP
  • Selecione a competência (mês/ano) dos dados
  • Faça upload do arquivo (.xlsx)
  • Os dados de exposição de RF serão disponibilizados na tela de Asset Allocation

⚠️ Colunas esperadas no arquivo:

✓ Cód. Assessor
✓ Cód. conta
✓ Ticker
✓ Nome papel
✓ Custódia
✓ Vencimento
✓ indexador
✓ Taxa cliente

📊 Classificação de Indexadores:

Pós-fixado: % CDI, CDI +, LFT, Renda+, Selic
Inflação: IPCA
Prefixado: PRE, PRÉ
Internacional: DOLAR PTAX
